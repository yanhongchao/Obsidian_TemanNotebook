## 术语对照表

| 术语    | 全称                                                      | 含义                                | 负责人/内容                        |
| ----- | ------------------------------------------------------- | --------------------------------- | ----------------------------- |
| APQP  | Advanced Product Quality Planning<br>（产品质量先期策划）         | 产品从概念到量产的全过程结构化项目管理方法             | 负责整个APQP项目的策划、推进和协调           |
| PPAP  | Production Part Approval Process<br>（生产件批准程序）           | 量产前向客户提交证明，证实过程能持续生产合格产品          | 负责准备和提交PPAP文件包，获得客户批准         |
| FMEA  | Failure Mode and Effects Analysis<br>（失效模式与影响分析）        | 预防性风险分析工具，识别潜在失效并提前采取措施           | 负责DFMEA（设计失效分析）和PFMEA（过程失效分析） |
| MSA   | Measurement System Analysis<br>（测量系统分析）                 | 评估测量设备和方法的准确性、精确性                 | 分析量具的重复性、再现性（GR&R），确保测量数据可靠   |
| CAMDS | China Automotive Material Data System<br>（中国汽车材料数据系统）   | 中国汽车行业材料数据申报平台，用于有害物质合规管理         | 在系统中录入材料数据，满足ELV等法规要求         |
| DV/PV | Design Verification / Process Validation<br>（设计验证/过程验证） | DV：验证设计是否符合规范<br>PV：验证过程能否稳定生产合格品 | 制定验证计划，执行测试并出具报告              |
| 生产工艺  | Production Process                                      | 制造流程设计、工装夹具、产线布局、作业指导书等           | 设计生产工艺流程，编制SOP，规划产线           |
| DFMEA | Design FMEA<br>（设计失效模式与影响分析）                     | 专注于产品设计阶段失效风险的分析方法                  | 识别设计中的潜在失效模式，评估风险，提出预防措施    |
| PFMEA | Process FMEA<br>（过程失效模式与影响分析）                     | 专注于制造过程失效风险的分析方法                     | 识别生产过程中的潜在失效模式，评估风险，采取控制措施 |
| SPC   | Statistical Process Control<br>（统计过程控制）                 | 使用统计方法监控和过程变异，确保过程稳定受控           | 使用控制图监控过程能力，识别特殊原因变异       |
| RPN   | Risk Priority Number<br>（风险优先数）                       | 失效模式的严重度×频度×探测度的乘积                  | 评估失效风险大小，确定改进优先级           |
| AP    | Action Priority<br>（行动优先级）                          | FMEA中使用的新优先级评估方法（基于S和O的组合）         | 更精确地确定行动优先级，聚焦高风险项目        |
| KPC   | Key Product Characteristic<br>（关键产品特性）                 | 对产品功能、安全或法规有重大影响的特性               | 制定特殊控制计划，确保这些特性的稳定受控       |
| PQC   | Product Quality Control<br>（产品质量控制）                  | 确保产品符合要求的所有活动和方法                     | 包括检验、测试、控制计划等活动             |
| Cp/Cpk| Process Capability Index<br>（过程能力指数）                   | Cp：潜在过程能力<br>Cpk：实际过程能力（考虑中心偏移） | 评估过程满足规格要求的能力，指导过程改进       |
| GRR   | Gauge R&R<br>（量具重复性与再现性）                         | 评估测量系统变异的方法                           | 确保测量数据可靠，用于过程决策             |
| SOP   | Standard Operating Procedure<br>（标准作业程序）            | 规定特定活动或操作的标准方法的文件                  | 确保操作的一致性和可追溯性，减少变异         |
| WI    | Work Instruction<br>（作业指导书）                          | 详细说明如何执行具体任务的文件                     | 为操作人员提供明确的执行步骤和要求           |
| QMS   | Quality Management System<br>（质量管理体系）                | 组织用于管理质量目标的体系框架                      | 建立和维持符合IATF16949的质量管理体系       |
| CTF   | Critical to Function<br>（功能关键特性）                     | 对产品功能实现至关重要的特性                       | 实施特殊的控制措施确保这些特性的稳定         |
| DOE   | Design of Experiments<br>（试验设计）                       | 系统地设计试验以分析因素对结果影响的方法              | 优化参数设计，提高产品质量和可靠性           |
| PPM   | Parts Per Million<br>（百万分之一）                        | 质量水平的度量单位，表示每百万个产品中的不合格品数量      | 设定质量目标，监控长期过程绩效             |
| MA    | Major Nonconformity<br>（主要不符合项）                    | 严重影响体系有效性的不合格                        | 需在规定时间内整改并验证有效性              |
| MI    | Minor Nonconformity<br>（次要不符合项）                    | 不影响体系整体有效性的孤立不合格                    | 需在规定时间内整改，一般无期限要求           |
| OEE   | Overall Equipment Effectiveness<br>（设备综合效率）           | 衡量设备生产效率的综合指标（可用率×性能×良品率）       | 识别设备损失，实施持续改进                |
