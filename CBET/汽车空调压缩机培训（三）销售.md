# 汽车空调压缩机培训（三）销售

## 一、汽车空调系统概述

### 1.1 系统功能
汽车空调系统是实现对车厢内空气进行制冷、加热、换气和空气净化的装置。

### 1.2 系统作用
- 调节车厢内的温度
- 调节车厢内的湿度
- 调节车厢内的空气流动速度
- 过滤尘埃

### 1.3 主要部件
- 压缩机
- 冷凝器
- 储液罐
- 膨胀阀
- 蒸发器
- 连接管路

### 1.4 工作原理
- **发动机舱内**：制冷剂由气态变为液态，放出热量
- **车内**：制冷剂由液态变为气态，吸收热量，降低车内温度

## 八、噪声

### 8.1 NVH概念
**NVH**：噪声（Noise）、振动（Vibration）、声振粗糙度（Harshness）

### 8.2 电动压缩机噪声来源

| 类型 | 来源 |
|------|------|
| **机械噪声** | 轴承运转、动静盘啮合摩擦 |
| **气流噪声** | 吸气气流、排气气流、排气孔周围气体膨胀 |
| **电磁噪声** | 磁场转换切割磁力线、IGBT开闭 |

### 8.3 噪声测试注意事项
压缩机工况、转速、探头距离相同的情况下才能比较分贝值：
- 压缩机工况有行标、国标等，排气压力越高噪音越大
- 测量距离通常为15cm、30cm、1m

### 8.4 噪声的传递
压缩机噪声除通过空气直接传入乘员舱外，还可以通过吸气管路、排气管路、车身等传入。

有时噪声仪测量的分贝数不高，但是人体感觉噪声大，可能的原因是在某个频率上噪声特别大，而人体对此频率很敏感。

例如：控制器的载波频率如果低于12k，人耳会感觉到明显的刺耳的声音，但是用噪声仪测量噪声值不高。

## 九、振动

### 9.1 振动三要素
- **幅值**
- **频率**
- **相位**

### 9.2 振动测量要素
- **振幅**
- **速度**
- **加速度**

### 9.3 振动比较方法

| 频率范围 | 比较参数 |
|----------|----------|
| 低频（<10Hz） | 振幅 |
| 中频（10～1000Hz） | 速度 |
| 高频（>1000Hz） | 加速度 |

压缩机振动属于**中频振动**，压缩机振动比较要基于相同的测试工装和测点布置位置。

### 9.4 振动标准（JB/12845-2016）

在转速允许范围内，亚临界用电动压缩机水平前后（X轴）、水平左右（Y轴）、垂直（Z轴）三个方向的实测激振力不超过公式计算值：

**Fmax = k × n**

其中：
- Fmax：最大允许激振力，单位为牛（N）
- K：激振系数，取0.02N·min/r
- n：压缩机转速值，单位为转每分（r/min）

## 十二、压缩机的控制方式

### 12.1 ON/OFF + PWM控制

#### ON/OFF启停
- 通过判断信号端的P2针脚电压状态确定压缩机是否工作
- 常用规则：**低电平（接地）启动/悬空停止**
- 也可以改为：高电平启动/悬空停止

#### PWM调速
- 通过判断信号端的P3针脚的高电平占空比确定压缩机转速
- 常用规则：高电平/悬空最高转速，低电平最低转速，中间占空比调速
- 注意：PWM调速信号容易受到干扰导致压缩机转速不稳

### 12.2 PWM调速计算

**转速计算公式**：
```
n(rpm) = B + (A - B) × X
```
其中：
- X：PWM百分比
- A：最高转速
- B：最低转速

**举例**（BET27-C901压缩机）：
- 最低转速：1000rpm
- 最高转速：6000rpm
- 占空比60%，实际转速为4000rpm

### 12.3 CAN2.0B或LIN2.1控制

#### 特点
CAN总线或LIN总线可以**双向传输信息**：
- VCU可以向控制器发送指令
- 控制器也可以向VCU上报工作状态

#### VCU发送指令
- 启动/停止
- 工作转速
- 最大允许电流等

#### 控制器上报信息
- 工作状态（启动/停止）
- 实际转速
- 当前电压
- 当前电流
- 当前故障（欠压/过压/过流/超温等）

#### 优势
- CAN控制还可以做**诊断功能**和**在线软件更新**
